Lista ordenada começa com ol por exemplo: 
<ol>
    <li>Facebook</li>
    <li>Instagram</li>
    <li>Twiter</li>
</ol>


Lista não ordenada começa com ul por exemplo:
<ul>
    <li>Facebook</li>
    <li>Instagram</li>
    <li>Twiter</li>
</ul> 


Lista de descrição começa com dl por exemplo: 
<dl>
    <dt>Facebook</dt>
    <dd>Facebook é uma rede social muito utilizada no mundo</dd>
    <dt>Instagram</dt>
    <dd>Instagram é rede social que está crecendo muito</dd>
    <dt>Twiter</dt>
    <dd>Twiter é outra rede social</dd>
</dl>


Descrição e feito pela tag dd exemplo do resultado:
Facebook
	Facebook é uma rede social muito utilizada no mundo
Instagram
	Instagram é rede social que está crecendo muito
Twiter
	Twiter é outra rede social

Tabelas
Para criar uma tabela utiliza a tag table exemplo:
<table>
    <caption> Cadastro de clientes</caption>
</table>

Uma tabela e constituinda por tag e linha e para isso utilizanmos a tag abaixo:
tr = linha
th = titulo 
td = conteudo
<table>
    <caption> Cadastro de clientes</caption>

    <tr>
        <th>Nome:</th>
        <th>Email:</th>
        <th>Telefone</th>
    </tr>

    <tr>
        <td>Gabriel</td>
        <td>gabrieljorge0027@gmail.com</td>
        <td>(31 98260-3399)</td>
    </tr>

 
</table>
colspan="" = Ocupar o total de colunas que eu desejar
rowspan="" = Ocupar o quanto de linhas eu desejar
colgroup: Serve para mim formatar a coluna
thead = Cabeçalho
tbody = corpo 
tfoot = Rodape
