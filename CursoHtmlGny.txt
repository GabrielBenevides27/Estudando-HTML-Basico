Lista ordenada começa com ol por exemplo: 
<ol>
    <li>Facebook</li>
    <li>Instagram</li>
    <li>Twitter</li>
</ol>


Lista não ordenada começa com ul por exemplo:
<ul>
    <li>Facebook</li>
    <li>Instagram</li>
    <li>Twitter</li>
</ul> 


Lista de descrição começa com dl por exemplo: 
<dl>
    <dt>Facebook</dt>
    <dd>Facebook é uma rede social muito utilizada no mundo</dd>
    <dt>Instagram</dt>
    <dd>Instagram é rede social que está crescendo muito</dd>
    <dt>Twitter</dt>
    <dd>Twitter é outra rede social</dd>
</dl>


Descrição e feito pela tag dd exemplo do resultado:
Facebook
	Facebook é uma rede social muito utilizada no mundo
Instagram
	Instagram é rede social que está crescendo muito
Twitter
	Twitter é outra rede social

Tabelas
Para criar uma tabela utiliza a tag table exemplo:
<table>
    <caption> Cadastro de clientes</caption>
</table>

Uma tabela e constituída por tag e linha e para isso utilizamos a tag abaixo:
tr = linha
th = titulo 
td = conteúdo
<table>
    <caption> Cadastro de clientes</caption>

    <tr>
        <th>Nome:</th>
        <th>Email:</th>
        <th>Telefone</th>
    </tr>

    <tr>
        <td>Gabriel</td>
        <td>gabrieljorge0027@gmail.com</td>
        <td>(31 98260-3399)</td>
    </tr>

 
</table>
colspan="" = Ocupar o total de colunas que eu desejar
rowspan="" = Ocupar o quanto de linhas eu desejar
colgroup: Serve para eu formatar a coluna
thead = Cabeçalho
tbody = corpo 
tfoot = Rodapé


O que é uma DIV:
Uma div é tipo um bloco que serve para colocar outros arquivos HTML exemplo: 
<div>
        <h1>Aprendendo o que é DIV</h1>
        <p>Lorem ipsum dolor sit amet consectetur adipiscing. </p>
</div>
Span= serve para modificar uma parte específica que se encontra dentro dela

Para adicionar um comentário no código utilizamos <!--

Meta: são descrições que utilizamos no início do código no head
<meta name="description" content="Curso de desenvolvimento"
<meta name="keywords" content="HTML5, CSS3">
<meta name="author" content="Gabriel Benevides"
Essa meta serve para atualizar a página no tempo que você determina
<meta http-equiv="refresh" content=""10


Para inserir Áudio utilizamos:
<audio controls>
        <source src="caminho"type="audio/mpeg">
</audio>
controls + autoplay = executar arquivo automático no iniciar da página
controls + autoplay + loop = áudio fica repetindo
