Lista ordenada começa com ol por exemplo: 
<ol>
    <li>Facebook</li>
    <li>Instagram</li>
    <li>Twitter</li>
</ol>


Lista não ordenada começa com ul por exemplo:
<ul>
    <li>Facebook</li>
    <li>Instagram</li>
    <li>Twitter</li>
</ul> 


Lista de descrição começa com dl por exemplo: 
<dl>
    <dt>Facebook</dt>
    <dd>Facebook é uma rede social muito utilizada no mundo</dd>
    <dt>Instagram</dt>
    <dd>Instagram é rede social que está crescendo muito</dd>
    <dt>Twitter</dt>
    <dd>Twitter é outra rede social</dd>
</dl>


Descrição e feito pela tag dd exemplo do resultado:
Facebook
	Facebook é uma rede social muito utilizada no mundo
Instagram
	Instagram é rede social que está crescendo muito
Twitter
	Twitter é outra rede social

Tabelas
Para criar uma tabela utiliza a tag table exemplo:
<table>
    <caption> Cadastro de clientes</caption>
</table>

Uma tabela e constituída por tag e linha e para isso utilizamos a tag abaixo:
tr = linha
th = titulo 
td = conteúdo
<table>
    <caption> Cadastro de clientes</caption>

    <tr>
        <th>Nome:</th>
        <th>Email:</th>
        <th>Telefone</th>
    </tr>

    <tr>
        <td>Gabriel</td>
        <td>gabrieljorge0027@gmail.com</td>
        <td>(31 98260-3399)</td>
    </tr>

 
</table>
colspan="" = Ocupar o total de colunas que eu desejar
rowspan="" = Ocupar o quanto de linhas eu desejar
colgroup: Serve para eu formatar a coluna
thead = Cabeçalho
tbody = corpo 
tfoot = Rodapé


O que é uma DIV:
Uma div é tipo um bloco que serve para colocar outros arquivos HTML exemplo: 
<div>
        <h1>Aprendendo o que é DIV</h1>
        <p>Lorem ipsum dolor sit amet consectetur adipiscing. </p>
</div>
Span= serve para modificar uma parte específica que se encontra dentro dela

controls + autoplay + loop = áudio fica repetindo

Para deixar um campo obrigatório apenas adicionar required exemplo:
<input type="text" name="nome" required><br>

Para colocar um focus apenas adicionar atributo autofocus exemplo:
<input type="text" name="nome" required autofocus><br>

Para deixar um campo desabilitado adicionamos o disabled exemplo:
<input type="text" name="nome" required autofocus disabled><br>

Para limitar o número de caractere que utilizamos em um campo utilizamos o maxlength="Numero" exemplo:
<input type="text" name="nome" required autofocus maxlength="5"><br>

Radio e serve para o usuário escolher uma dentre outras opções exemplo:
<input type="radio" name="time" value="cruzeiro">Cruzeiro<br>
<input type="radio" name="time" value="atletico">Atletico<br>
<input type="radio" name="time" value="america">America<br>

checkbox o usuário pode escolher quantas opção desejar exemplo: 
<input type="checkbox" name="cores" value="azul">Azul<br>
<input type="checkbox" name="cores" value="verde">Verde<br>
<input type="checkbox" name="cores" value="amarelo">Amarelo<br>

Para definir um padrão basta adicionar o checked exemplo:
<input type="radio" name="time" value="cruzeiro" checked>Cruzeiro<br>

Para colocar um campo para números podemos utilizar o type number exemplo:
<input type="number" name="quantidade" min="0" max="20">

Para utilizar um slide para informa o valor utilizamos o type range.

Para adicionar um campo para o usuário escolher uma cor utilizamos o type Color.

Para colocar algumas opções para completar automático utilizamos autocomplete="on" =ligado "off" Desligado.

Para colocar um texto dentro do input utilizamos o placeholder exemplo:
<input type="text" name="nome" required autofocus placeholder="Informe seu Nome"><br>
